# Vitality Herbs - Premium Chinese Health Supplements

A modern e-commerce website for selling Chinese herbal health products.

## Features
- 🌿 Responsive design
- 🛒 Shopping cart functionality
- 💳 Ready for payment integration
- 📱 Mobile-friendly

## Deployment
This site is deployed on GitHub Pages.

## Payment Integration
To add real payment processing, integrate with:
- Stripe (https://stripe.com)
- PayPal (https://paypal.com)

Both offer pay-per-transaction pricing with no monthly fees.
